# PocketLAN Android

PocketLAN Android is a standalone Android app that shares a user-selected device folder over HTTP to browsers on the same LAN.

## Features

- Native Material control panel
- Android Storage Access Framework folder picker
- Foreground service while sharing
- LAN URL discovery
- Optional browser PIN
- Self-contained browser file explorer
- Browse, search, preview, download, upload, create folder, rename, and delete

## Build

Open this folder in Android Studio, or build from a terminal with an Android SDK installed:

```powershell
gradle :app:assembleDebug
```

The APK output is written to `app/build/outputs/apk/debug/`.
